Conceito geral

A especificidade é uma forma de determinar qual regra CSS será aplicada a um elemento quando existem regras conflitantes. Ela é calculada com base em diferentes componentes, como seletores de ID, classes, atributos, pseudo-classes, seletores de elemento e pseudo-elementos. Cada componente contribui para a contagem da especificidade, onde valores mais altos indicam maior precedência.

Seletor de ID

O seletor de ID seleciona elementos com base em seus IDs únicos. Ele tem um peso maior do que os outros tipos de seletores, o que significa que uma regra com um seletor de ID terá precedência sobre as outras.

  • Exemplos
    1. #header Seleciona o elemento com o ID "header".
    2. #sidebar Seleciona o elemento com o ID "sidebar".
    3. #content Seleciona o elemento com o ID "content".

Seletor de classe

O seletor de classe seleciona elementos com base nas classes atribuídas a eles. Ele tem um peso médio em termos de especificidade.

  • Exemplos
    1. .destaque Seleciona os elementos que possuem a classe "destaque".
    2. .botao Seleciona os elementos que possuem a classe "botao".
    3. .menu-item Seleciona os elementos que possuem a classe "menu-item".

Seletor de atributo

O seletor de atributo seleciona elementos com base em atributos específicos. Ele tem um peso médio em termos de especificidade.

  • Exemplos
    1. [href] Seleciona os elementos que possuem o atributo href definido.
    2. [type="text"] Seleciona os elementos que possuem o atributo type definido como "text".
    3. [data-info] Seleciona os elementos que possuem o atributo data-info definido.

Seletor pseudo-classe

O seletor pseudo-classe seleciona elementos com base em estados ou interações específicas. Ele tem um peso médio em termos de especificidade.

  • Exemplos
    1. :hover Seleciona um elemento quando o cursor do mouse está sobre ele.
    2. :first-child Seleciona o primeiro elemento filho de seu pai.
    3. :nth-child(odd) Seleciona todos os elementos ímpares entre os elementos irmãos.

Seletor de elemento

O seletor de elemento seleciona elementos com base em seus nomes de tag. Ele tem o menor peso em termos de especificidade.

  • Exemplos
    1. div Seleciona todos os elementos <div> .
    2. p Seleciona todos os elementos <p>.
    3. a Seleciona todos os elementos <a>.

Seletor pseudo-elemento

O seletor pseudo-elemento seleciona partes específicas de um elemento. Ele tem o menor peso em termos de especificidade.

  • Exemplos
    1. ::before Seleciona o conteúdo inserido antes do elemento.
    2. ::after Seleciona o conteúdo inserido após o elemento.
    3. ::first-line Seleciona a primeira linha de texto dentro do elemento.
